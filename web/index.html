HOW IT WORKS

提醒规则

在节点编辑器中填写保号周期。达到周期后每天最多提醒一次；完成维护后在节点卡片点击“已保号”，周期会重新计算。

通知失败不会影响代理请求，也不会把 Bot Token 写入日志。

  1. 临时初始化

    安装后通过 http://服务器IP:9080/随机管理路径 完成管理员与 2FA 设置。

  2. 域名解析

    把域名 A/AAAA 记录指向当前服务器，确认 80/443 可访问。

  3. 执行 HTTPS 收口
    sudo aegis-relay domain example.com admin@example.com
  4. 最终状态

    脚本签发证书、启用自动续期、把 9080 重新绑定至 127.0.0.1；之后直接打开 HTTPS 域名根地址即可进入管理面板。
